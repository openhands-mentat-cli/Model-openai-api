Example Usage

                    
                    

                        
Python:

                        


import openai

client = openai.OpenAI(
    base_url="/v1",
    api_key="$$Hello1$$"
)

response = client.chat.completions.create(
    model="phi-3-mini-128k",
    messages=[
        {"role": "user", "content": "Hello!"}
    ],
    max_tokens=500
)

print(response.choices[0].message.content)
